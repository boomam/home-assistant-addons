## About
Newt is a fully user space WireGuard tunnel client and TCP/UDP proxy, designed to securely expose private resources controlled by Pangolin. By using Newt, you don't need to manage complex WireGuard tunnels and NATing.  

[Click here for the full Newt documentation](https://docs.pangolin.net/manage/sites/understanding-sites)  

## Configuration
During configuration, make sure you are adding *all* of the environment variables asked for.
You can gather most of these through following the Pangolin Docs on the topic, [here](https://docs.pangolin.net/manage/sites/install-site).
